🧪 ReqRes API Test Automation

Proyecto de automatización de pruebas para la API pública de ReqRes enfocado en validar el comportamiento de servicios REST mediante buenas prácticas de testing y diseño limpio.

Este proyecto implementa el patrón Screenplay con Serenity BDD, permitiendo construir pruebas escalables, legibles y mantenibles.

🛠️ Tecnologías utilizadas
    Java
    Rest Assured
    Serenity BDD
    Cucumber (BDD)
    Gradle
    JUnit
    
🧩 Patrón de diseño

Se implementa el patrón Screenplay, lo que permite:

Separar claramente responsabilidades
Reutilizar lógica de negocio
Mejorar la legibilidad de los tests
